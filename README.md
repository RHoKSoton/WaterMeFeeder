# How to use  #
---------------

* Dependancies
	- numpy
	- gdal ( work in progress )


* Open the script
* Pick a particular month and year and modify the values in 
	`aMonth`
	`aYear`

* Run the script with 
	`>>> python modis_gdal.py`
